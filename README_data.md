# Data files

Notebook này cần các file sau:

- `website_sessions.csv`
- `website_pageviews.csv`
- `orders.csv`
- `products.csv`

## Cách dùng trên Google Colab

1. Mở notebook bằng Google Colab.
2. Upload các file CSV vào thư mục `/content/`.
3. Chọn **Runtime → Run all**.

## Data sharing

Không đưa dữ liệu khách hàng, dữ liệu công ty hoặc dữ liệu có điều khoản hạn chế lên public repository. Nếu dataset không được phép công khai, chỉ giữ file hướng dẫn này và mô tả nguồn dữ liệu trong README chính.
